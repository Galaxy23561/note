# Git 基础

## Git

Git 是一个版本控制系统，允许您跟踪文件的变化。它被软件开发人员用于管理源代码和协作项目。Git 是一个分布式版本控制系统，这意味着整个代码库和历史记录都可以在每个开发人员的计算机上使用，这使得分支和合并变得容易。

## GitHub

GitHub 是一个基于 Web 的 Git 存储库托管服务。它提供了基于 Web 的图形界面、访问控制和几个协作功能，如 bug 跟踪、功能请求、任务管理和每个项目的维基。

## GitLab

GitLab 是一个基于 Web 的 Git 存储库管理器，具有维基和问题跟踪功能。它类似于 GitHub，但 GitLab 是开源的，可以自行托管。

## Bitbucket

Bitbucket 是一个基于 Web 的 Git 和 Mercurial 存储库托管服务。它类似于 GitHub，但 Bitbucket 对小团队免费，并提供无限的私有存储库。

## GitKraken

GitKraken 是一个跨平台的 Git 客户端，具有图形界面。它免费供非商业使用，并支持 GitLab、GitHub 和 Bitbucket。

## Git Bash

Git Bash 是 Git 的命令行界面。它包含在 Git for Windows 中，并提供用于从命令行运行 Git 的 Bash 模拟。

## Git GUI

Git GUI 是 Git 的图形用户界面。它包含在 Git for Windows 中，并提供用于从命令行运行 Git 的图形界面。

## Git Extensions

Git Extensions 是 Git 的图形用户界面。它是免费和开源的，并提供用于从命令行运行 Git 的图形界面。
